Pix4d Captur

É unha das aplicacións máis coñecidas para a planificación de voo. Trátase dunha aplicación móbil dispoñible para Android e para iOS.

É compatible coa maioría de drons profesionais máis comercializados (phantom, inspire, mavic, bebop 2, 3dr só…).
Vindo de parte de Pix4D, podedes esperar unha integración perfecta cos diferentes sistemas de software do fabricante.

O funcionamento desta aplicación é realmente sinxelo. Unha vez que teñamos clara a zona que queremos mapear, abrimos a aplicación e seleccionamos a área a sobrevoar. Como noutras apps, é moi sinxelo configurar aspectos relativos ao noso voo: velocidade, ángulo de inclinación da cámara ou a porcentaxe de solape entre imaxes.

Unha vez definido e calculados os roteiros, a aplicación comunicarase co noso dron e este empezará o voo de forma autónoma.

Esta app permite definir voos en forma de cuadrícula, dobre cuadrícula (solapando dúas cuadrículas perpendiculares), voos circulares e mesmo unir varios dos anteriores dentro dun único voo. Con isto están aseguradas as reconstrucións tanto de mapas planos, volumes e obxectos tridimiensionais.
